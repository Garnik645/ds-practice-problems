## Алгоритмические задачи с использованием Set и Map

## Руководство по работе с CSES

### Что такое CSES?

CSES (Code Submission Evaluation System) — это онлайн-платформа для решения алгоритмических задач.
Сайт содержит коллекцию из более чем 300 задач различной сложности, охватывающих основные темы алгоритмов и структур данных.

- Официальный сайт CSES: https://cses.fi

### Автоматическое тестирование

Все задачи CSES имеют множество тестов и каждое решение может автоматически проверяться на этих тестах.
Основное ограничение — это лимит времени выполнения программы (обычно 1-2 секунды).
То есть ваше решение должно дать правильный ответ при любых входных данных за этот лимит времени.
**Перед отправкой решения обязательно рассчитайте сложность вашего алгоритма**, чтобы понять, пройдёт ли оно по времени.

Например, если в задаче указано, что n ≤ 200.000, то алгоритм O(n²) получит **Time Limit Exceeded**, а O(n log n) пройдёт успешно.
Используйте структуры данных `std::set` и `std::map` именно для того, чтобы улучшить асимптотическую сложность решения.

### Возможные вердикты тестирования

- **Accepted (AC)** — решение правильное
- **Wrong Answer (WA)** — неправильный ответ
- **Time Limit Exceeded (TLE)** — превышено ограничение по времени
- **Runtime Error (RE)** — ошибка выполнения программы
- **Compile Error** — ошибка компиляции

### Регистрация и начало работы

1. Перейдите на сайт CSES
2. Создайте аккаунт, нажав "Login" -> "Create Account"

### Как отправлять решения

- Нажмите "Submit"
- Загрузите файл с вашим решением
- Выберите язык программирования (C++17 или C++20)
- Дождитесь результата проверки (обычно несколько секунд)
